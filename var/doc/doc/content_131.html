Description

This command loops over its body while the value of its test expression is true.

Command Syntax

'@x:while' <exp>

No options are permitted.

Execution Semantics

The test expression is evaluated and must yield a boolean result. If this result is false, then execution of the @x:while command is complete. If this result is true, then the body is executed and then the entire command is executed again. The result of the @x:while command will be the result of applying Join Rules to its body executions.

Examples


   @x:while i < 10 {
      g(i);
      i := f(i)
   }
