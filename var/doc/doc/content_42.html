@x:Act has a rich set of facilities for parsing (Parse), processing (xdom), and unparsing (Unparse) XML.

Since @x:Act allows multiple XML elements at the top level, such XML files can be easily be used for log files. The @x:Act interpreter logs use this approach.

XML reserves several special character including &, <, and >. The quoteXML function takes a string and quotes these special characters. When @x:Act is producing XHTML output for display in a web browser, this quoting allows arbitrary strings to displayed. This is particularly useful when outputting debugging output.
