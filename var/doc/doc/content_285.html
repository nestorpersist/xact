Join rules determine the result of a sequence of expressions. Consider for example.

@x:block {
   a;
   b
}

The result of the block will be the result of joining the value of a to the value of b. If one value if null then the joined result is the other value. If both values are strings, then the result is a ++ b.

For more information see Join Rules.
