This function takes a string and escapes it using XML escapes so it contains only ASCII characters.

Function asciiXML

@x:func asciiXML(s~string)~string
s a string value.

Semantics

The escaped string. Non-breaking space (nbsp) is escaped to &nbsp;. Characters less then 32 (except cariage return and line feed) and characters greater than or equal to 127 (other than non-breaking space) are escaped using XML hex <escape>s. The asciiXML option applies this same set of escapes to program output.
