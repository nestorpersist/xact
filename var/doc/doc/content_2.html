@x:Act (pronounced "exact") is a powerful scripting language with strong support for XML and Java.

This documentation is divided into several major sections.

  • Introduction. Continue in this section for a short introduction to @x:Act.
  • User Manual. This section provides a detailed discussion of all the major @x:Act features.
  • System Manual. This section describes how to install, configure, and run @x:Act
  • Reference. This section contains a detailed reference of all @x:Act features.
  • Libraries. This section describes the included @x:Act library scripts that that provide a number of useful features.

@x:Act examples are a great way to learn @x:Act and better understand how to use it more advanced features. Running examples (particularly by single stepping them via the debugger) can be helpful. @x:Act example code can be found in the following places.

  • Introduction Examples. These simple example are included in the Introduction.
  • Reference Examples. A collection of more advanced examples are included in the Reference section.
  • Libraries. Source code of the libraries provides yet another set of examples.

See the Support section for support information.
