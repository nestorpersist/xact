This function combines a sequence of strings from an iterator using a specified separator string.

For example the result of combining iterator Vals["a","b","c"] using separator string "," is "a,b,c".

Function Combine[ ]

@x:func Combine[iter,sep~string]~string
iter an iterator over a sequence of strings.
sep a separator.

Semantics

The result of combining all iterator iter string values using the specified separator sep.
