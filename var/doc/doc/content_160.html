View Function sys:Is

@x:func sys:Is(v~boolean)~boolean
v value to test

Semantics

The result is v?string.

View Function sys:Make

@x:func sys:Make(s~string)~fold
s a string value

Semantics

The resulting string will be the same as s except all upper-case letters will have been folded to lower-case.

View Function sys:Eq

@x:func sys:Eq(a~string,b~string)~boolean
a first parameter
b second parameter

Semantics

After folding all upper case letters to lower case, if a and b have the same size and same characters, then the result is true, else the result is false.

View Function sys:Ne

@x:func sys:Ne(a~string,b~string)~boolean
a first parameter
b second parameter

Semantics

After folding all upper case letters to lower case, if a and b have the same size and same characters, then the result is false, else the result is true.

View Function sys:Less

@x:func sys:Less(a~string,b~string)~boolean
a first parameter
b second parameter

Semantics

After folding all upper case letters to lower case, if a is less than b, then the result is true, else the result is false.

View Function sys:LessEq

@x:func sys:LessEq(a~string,b~string)~boolean
a first parameter
b second parameter

Semantics

After folding all upper case letters to lower case, if a is less than or equal to b, then the result is true, else the result is false.
