Description

The expression
   a..b
is desugared to
   Range(a,b)
which in turn invokes
   Range[a,b]
which is an iterator function that iterates over all int values starting with a and ending with b.

Function Range

@x:func Range(first~int,last~int)
first first value.
last last value.

Semantics

The result is the iterator Range[first,last].

Function Range[ ]

@x:func Range[first~int,last~int](act)
first first value.
last last value.
act the iterator body function. It takes a single parameter of type int.

Semantics

This is an iterator function that iterates over all int values between a and b inclusive. If b < a, then the iterator body act is not executed.

Examples


   @x:var result := 1;
   @x:for i := 1..10 {
      result := result * i
   }
