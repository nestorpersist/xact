This iterator takes two other iterators and runs the second nested inside the first.

Function Nest[ ]

@x:func Nest[iter1~func,iter2~func](act)
iter1 first iterator function.
iter2 second iterator function.
act the iterator body function. It takes a two parameters, the first for iter1 and the second for iter2.

Semantics

This is an iterator function that iterates over iter1 and for each iter1 value iterates over all iter2 values.
