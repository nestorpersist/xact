This iterator takes an iterator and then iterates over all its values except the first.

Function ExceptFirst[ ]

@x:func ExceptFirst[iter~func](act)
iter iterator function.
act the iterator body function. It takes a one parameter.

Semantics

This is an iterator function that iterates over over all values of iter except the first. If iter has one value or no values, then act is not executed.
