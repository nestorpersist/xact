@x:Act commands need to know where to find various software upon which they depend.

On Windows copy xact/dev/win/jrndef.bat to xact/def.bat. You should then edit xact/def.bat to have the correct paths for your machine.

On a Macintosh copy xact/dev/mac/macdef to xact/def. You should then edit xact/def to have the correct paths for your machine.

  • XACT. This should be the path for where you installed @x:Act.
  • JAVA_HOME. This should point to the Java JDK. Version should be at least 1.5.
  • ANT_HOME. Directory where ANT is installed. You can get ANT from.

    http://ant.apache.org/

  • BDB_JAR. This should point to the jar file for Berkeley DB. You want the Java API on the standard edition (not the Java edition). You can get this from.

    http://ant.apache.org/

  • SERVLET_JAR. This should point to a servlet Jar. We run @x:Act under Tomcat which includes this jar file. You can get Tomcat from.

    http://tomcat.apache.org/
