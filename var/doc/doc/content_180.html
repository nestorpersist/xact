Description

The expression
   a > b
is desugared to
   Greater(a,b).

Function Greater

@x:func Greater(a,b)~boolean
a first parameter
b second parameter

Semantics

If Cover(Type[a],Type[b]) is null, then the result is
   false
else the result is
   Cover(Type[a],Type[b]).sys:Greater(a,b)
