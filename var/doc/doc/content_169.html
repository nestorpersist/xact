This function calls the interpreter bind1 and bind2 passes on a specified tree. Bind1 processes scopes and name definitions. Bind2 processes name uses.

Function Bind

@x:func Bind(tree~xdom,language := lang:xact)~void
tree the tree to be bound.
language (optional) The language to be used for binding.

Semantics

Scopes, name definitions, and name uses are bound on the specified tree using the specified language.
