Description

View of xdom:element for attribute lists. List values are stored as follows.

xdom value directly stored as is
string value stored in a xdom:string node
int value stored in a xdom:int node
float value stored in a xdom:float node
any other value stored in a xdom:value node

View list

@x:view list

@x:view @x:func sys:Is(v~boolean)~boolean

View Function sys:Is

@x:func sys:Is(v~boolean)~boolean
v value to test

Semantics

The result is v?xdom:element.

Function sys:Make

@x:func sys:Make(%L)~list
%L a list of named or unnamed values

Semantics

The result is a new list constructed from the specified list of values. The tag of the resulting xdom:element is "list".
