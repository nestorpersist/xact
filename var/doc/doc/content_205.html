Description

This special function should only be called within Language Definition types. It walks a subtree of type xdom which is not a xdom:element. The DocLang example shows a sample use.

Function walk:exp

@x:func walk:exp(exp~xdom)
exp the subtree to be walked. Must not have type xdom:element.

Semantics

The pass result. This will be null for bind1 and bind2 and the value produced by evaluating the expression in exec.
