A Java object can be created by calling a constructor for a Java class. Unlike Java, no new keyword is needed. For example @x:const jfile := Java.java.io.File(fname) creates a new File object;

Object methods can be called in the usual way. For example, jfile.delete().

The java and JavaProp examples show examples of calling Java code.
