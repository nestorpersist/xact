Syntax
<element> ::= <start tag> S? {<xml> S?}* <end tag>
<element> ::= <empty tag>
<element> ::= '<?xml' {S <attr>}* S? '?>'
<element> ::= '<' {<short cmd> | <long cmd>} S? '/>'

Description

An XML element.

Note how a tagged short or long command can be embedded inside XML using this escape sequence.

Parsing

For a simple XML element, result will be an xdom:element tree node with kind "xml".

For the ?xml form, the result will be an xdom:element tree node with tag @xml and kind xmldecl.
