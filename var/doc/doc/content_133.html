Description

The type func is the type of all functions. Functions are declared using the @x:func tag. Functions are called using the call form of <qualified> expressions.

Type func

@x:type func

@x:type @x:func sys:Eq(a~func,b~func)~boolean
@x:type @x:func sys:Ne(a~func,b~func)~boolean
@x:self @x:func toString()~string

Type Function sys:Eq

@x:func sys:Eq(a~func,b~func)~boolean
a first parameter
b second parameter

Semantics

If a and b are the same function, then the result is true, else the result is false.

Type Function sys:Ne

@x:func sys:Ne(a~func,b~func)~boolean
a first parameter
b second parameter

Semantics

If a and b are the same function, then the result is false, else the result is true.

Self Function toString

@x:func toString()~string

Semantics

The result is the name of the function.

Examples


   @x:func incr(x) {x + 1}
   ~Type[incr] == func;
   incr(7)
