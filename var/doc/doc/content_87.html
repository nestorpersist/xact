Syntax
<extended id> ::= {letter | '_' | ':' | '.'}
{letter | '_' | ':' | '.' | '-' | digit}*

Description

Extended ids are the XML-like form of <id>.

Semantics

The case of letters in ids is significant.

Examples

Some extended ids in an XML context
   <a:b-c d.x="3"/>
An extended variable name in a non-XML context.
   @x:var $a:b.c$ := 0
