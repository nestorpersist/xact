Description

The expression
   a + b
is desugared to
   Add(a,b)

Function Add

@x:func Add(a,%L)
a first parameter
%L zero or more additional parameters

Semantics

If the call has form Add(a), then the result is
   a
else if the call has form Add(a,b), then the result will be
   Cover(Type[a],Type[b]).sys:Add(a,b)
else the call will have form Add(a,b,%M) and the result will be
   Add(Add(a,b),%M)
