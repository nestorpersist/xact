Description

View of string where the value is the path name of a file or directory.

Examples of how to use the file view can be found in the file example.

Self Function exists

@x:func exists()~boolean

Semantics

Result is true if the file exists.

Self Function create

@x:func create()~boolean

Semantics

Creates the file if it does not exist. Result is true if the file was created (this is, it did not exist prior to this call).

Self Function isDirectory

@x:func isDirectory()~boolean

Semantics

Result is true if the file is a directory.

Self Function read

@x:func read(charSet=charSet1~string := ISO - 8859 - 1)~string
charSet=charSet1 (optional) The Character Set to use for converting external bytes to internal string characters.

Semantics

Returns the contents of the file.

Self Function write

@x:func write(val~string,charSet=charSet1~string := ISO - 8859 - 1)~void
val a value to be written.
charSet=charSet1 (optional) The Character Set to use for converting internal string characters to external bytes.

Semantics

Replaces the contents of the file with val. It creates the file if it does not exist.

Self Function append

@x:func append(val~string,charSet=charSet1~string := ISO - 8859 - 1)~void
val a value to be appended.
charSet=charSet1 (optional) The Character Set to use for converting internal string characters to external bytes.

Semantics

Appends val to the end of the current contents of the file. It creates the file if it does not exist.

Self Function createDirectory

@x:func createDirectory()~void

Semantics

Creates a new directory.

Self Function delete

@x:func delete()~void

Semantics

Deletes the specified file or directory.

Self Function rename

@x:func rename(new~string)~void
new a new name for the file.

Semantics

Renames the file to the new name.

Self Const files

@x:const files~func

Semantics

Requires self.isDirectory(). An iterator function that iterates over the names of all files in a directory.
