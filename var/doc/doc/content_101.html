Syntax
<short cmd> ::= '@' <id> S {<exps>}?

Description

A list of expressions.

Examples


   f(1,2,3)

   Enum["red","green","blue"]

Description

A short command is a tagged command that does not have a body. The <id> is the tag. The <exp>'s are command attributes and options.

Parsing

The result of parsing a short command will be a xdom:element node with kind "normal" and an empty body.
