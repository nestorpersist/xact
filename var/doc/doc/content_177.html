Description

The expression
   a=b
is desugared to
   Equal(a,b)

Function Equal

@x:func Equal(a,b)~void
a a <id> that names a value.
b the value to be named

Semantics

The Equal function has special assignment semantics. Its use is restricted to certain special contexts. First it can be used for named parameters. Second it is used for named xdom:element attributes.
