To run an @x:Act <script>, you must know the script path for the file that contains the script. A script path is relative to the @x:Act scripts directory. By default this directory is at xact/var/scripts.

To run the script in xact/var/scripts/xxx.xact the script path will be xxx. To run the script in xact/var/scripts/xad/example/fact.xact the script path will be xad:example:fact. The following command will run this script.

run xad:example:fact
