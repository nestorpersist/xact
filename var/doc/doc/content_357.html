Description

This special function should only be called within Language Definition types. It walks an xdom:element tree node.

Function walk:elem

@x:func walk:elem(elem~xdom:element,lang)
elem the element to be walked.
lang The language object to be used to process the element.

Semantics

The pass result. This will be null for bind1 and bind2 and the value produced by executing the element in exec.
