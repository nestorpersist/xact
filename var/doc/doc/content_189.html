The expression
   a != b

   Ne(a,b).

Function Ne

@x:func Ne(a,b)~boolean
a first parameter
b second parameter

Semantics

If Cover(Type[a],Type[b]) is null, then the result is
   true
else the result is
   Cover(Type[a],Type[b]).sys:Ne(a,b)
