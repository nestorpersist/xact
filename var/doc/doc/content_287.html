@x:Act allows functions to appear on both the left-hand-side (LHS) and right-hand-side (RHS) of assignments. This is handy when defining abstractions such as dot qualifications and subscripting that are handle to use in both places. IT is also handy for referencing type data components. Unlike Java that requires separate getter and setter functions, in @x:Act they can be combined into a single LHS/RHS function. Consider a couple of examples.


   @x:var r := rec(a=3);
   @x:var a := array(1,2,3);
   r.a := r.a + 1;
   a(3) := 2 * a(3)

LHS/RHS functions are declared with two @x:func declarations for the same name, one with the lhs option and the other with the rhs option.

For more information see LHS and RHS.
