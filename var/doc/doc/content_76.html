Syntax
char ::= letter | digit | '@' | '_' | '.' | '$' | '%' |
| '(' | ')' | '{' | '}' | '[' | ']' | ''' | '"' | '`' |
| '+' | '-' | '*' | '/' | '|' | '^' | '&' | '!' |
| ':' | '=' | '<' | '>' |
| '~' | '?' | '#' | ';' | ',' | '\'

Description

These are the permitted printable characters. See also S, the set of permitted space characters.
