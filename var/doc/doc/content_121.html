Description

This command changes the language used to interpret following commands. It specifies a language object for the new language. For more information see the Language Definition section.

Command Syntax

'@x:lang' <id> ',' <id>

Semantics

The first attribute <id> will be a script path that identifies an external script. The second attribute <id> will be a visible name of that script. Evaluation of that name must yield a language object that will be used to interpret all code after the @x:lang command through the end of the smallest enclosing scope.
