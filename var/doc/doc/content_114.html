Description

This command work with an immediately preceding if, orif or for command. ITs body is executed only if the body of the previous command was not executed.

Command Syntax

'@x:else'

No attributes, options or body are permitted.

A @x:else command must directly follow a @x:if, @x:orif or @x:for command.

Execution Semantics

If the body of the previous command was executed, then the result of executing the @x:else command is null. If the body of the previous command was not executed, then the result of the @x:else command is the result of executing its body.

Examples


   @x:if i < j {
      "less"
   } @x:orif i > j {
      "greater"
   } @x:else {
      "equal"
   }

   @use library:iterator;
   @x:for i := First[Filter[iter,p]] {
      "found:"; i
   } @x:else {
      "not found"
   }
