This view is a convenient way of specifying relative script paths. Suppose that the current scripts as given by options.script#rec is a:b:c. Now REL.x:y returns "a:b:x:y". See also REL.

It is also possible to specify the path from which to be relative. For example REL["m:n"].x:y returns "m:n:x:y"

View REL

@x:view REL

@x:view @x:func sys:Dot[name~string]~string

View Function sys:Dot[ ]

@x:func sys:Dot[name~string]~string
name the relative script path

Semantics

The result is the corresponding absolute script path.
