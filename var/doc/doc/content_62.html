On Windows, connect to xact/dev/win.

  • To compile the @x:Act interpreter issue the following commands.

    ant.bat

    jar.bat

  • To build the @x:Act documentation issue the following command.

    run.bat xad:gen(doc)

  • To build the Javadoc issue the following command.

    javadoc.bat

On a Macintosh, connect to xact/dev/mac.

  • To compile the @x:Act interpreter issue the following commands.

    ./ant

    ./jar

  • To build the @x:Act documentation issue the following command.

    ./run "xad:gen(doc)"

  • To build the Javadoc issue the following command.

    ./javadoc
