Description

The xdom:name type is used to represent name nodes within an xdom tree.

Semantics

Initializes a new xdom:name tree node.

Self Var val

@x:var val~string

Semantics

The contained name.

Self Var space

@x:var space~string

Semantics

The part of the contained name before the first : or "" if there is no :.

Self Var base

@x:var base~string

Semantics

The part of the contained name after the first : or val if there is no :.

Self Var hasExt

@x:var hasExt~boolean

Semantics

The name has a name extension.

Self Const size

@x:const size~int

Semantics

The number of xdom tree nodes that are within the name extension.

LHS/RHS Self Function sys:Subscript

@x:func sys:Subscript(index~int)
index The index of the name extension item to be selected. For name node n this must be in the range 1..n.size.

Semantics

The indexth name extension item value.

Self Function insert

@x:func insert(val,pos~int := self.size)~void
val a value to be inserted
pos (optional) the value must be in the range 0..self.size.

Semantics

Inserts the new value at the specified position. If the position is 0, the value is inserted at the beginning of the name extension list. Otherwise, the value is inserted after the pos item of the list.

Self Function delete

@x:func delete(pos~int)~void
pos (optional) for name node n, the value must be in the range 1..n.size.

Semantics

The element at the specified position is deleted from the name extension list.

Self Const all

@x:const all~func

Semantics

An iterator function that iterates over the extension list.
