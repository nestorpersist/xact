When running a script function, it is possible to pass parameters to that function. All parameters must have type string. For example to pass "abc" and "123" to script function f use the following command.

run f(abc,123)

Some string character must be escaped. For example to pass "a b" and "<{>" to script function f use the following command.

run f(a+b,!(lt)!(=123)!(gt))
