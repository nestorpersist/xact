Even though @x:Act is dynamically typed, it is possible to specify types in a way similar to that of statically typed languages.


@x:func fact(n~int)~int {
   ~n >= 0;
   @x:if n == 0 {
      1
   } @x:else {
      n * fact(n - 1)
   }
}
@x:for i := 1..10 {
   @x:const j~int := fact(i);
   "fact(&i;)=&j;&eol;"~string
}

The infix ~ operator asserts that a name or expression has a specified type. The prefix ~ operator asserts that the following boolean expression is true.
