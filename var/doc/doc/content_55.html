The queue and stream types provides the main high-level ways to synchronize multiple threads.

For a queue object, one thread makes multiple calls to the send function and another thread makes multiple calls to the receive function. Each call sends or receives a single value.

For a stream object one thread makes multiple calls to the send function followed by a single call to the close function and another thread makes a single call to the receive function. Each send call sends a string and the receive call retrieves the single concatenated result of all the sends.
