Description

This special function should only be called during the bind1 pass within Language Definition types. It defines a a name in a scope. The DocLang example shows a sample use.

Function walk:define

@x:func walk:define(name~xdom:name,scope~xdom:element)~null
name the name to be defined.
scope the scope where the name is to be defined.

Semantics

The name name is defined in scope scope.
