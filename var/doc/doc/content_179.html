This function calls the interpreter exec pass on a specified tree. This pass executes the tree.

Function Bind

@x:func Bind(tree~xdom,language := lang:xact)
tree the tree to be executed.
language (optional) The language to be used for execution.

Semantics

The specified tree is executed using the specified language. The result of calling Exec will be the result of that tree execution.
