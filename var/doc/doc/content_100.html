Syntax
<script> ::= {S? <cmds>}? S?

Description

This is the top level non-terminal for the @x:Act syntax. Each @x:Act script lives in a separate file.

Parsing

When a script is parsed the sequence of commands are placed within the body of a single top-level xdom:element tree node with tag @x:block.

The Character Set used for scripts is "UTF-8".
