In addition to parameters, there is another mechanism for passing data to a script. The @icode{passed}/> record contains named values passed to the script. Although passed was designed to enable HTML form input to be passed to @x:Act scripts, it can be used for other purposes. Consider the following command line.

run "foo(abc)&a=true&b=123"

This sets the passed.a#rec and passed.b#rec to the specified string values.
