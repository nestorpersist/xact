This iterator takes an iterator with two indexes and a filtering function and iterates over all values that satisfy the filter.

Function Filter2[ ]

@x:func Filter2[iter~func,filter~func](act)
iter iterator function with two indexes.
filter filter function. Takes two parameters and returns a boolean result.
act the iterator body function. It takes a one parameter.

Semantics

This is an iterator function that iterates over all values [v1,v2] of iter where filter(v1,v2) returns true.
