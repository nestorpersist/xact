@x:Act has two alternative syntax forms: one form is XML and the other form is more Java-like. In many scripts a combination of both forms is used.

The XML form supports most XML features. The one feature not supported is DTDs. A useful generalization is that a <script> can consist of a sequence of XML tags (unlike XML where only a single top level tag is permitted).
