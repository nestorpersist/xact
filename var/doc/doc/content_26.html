@x:Act commands by default have tags with namespace x:.

There are 22 @x:Act command tags. For example, the command with tag @x:var is used to declare variables and the command with tag @x:while is a while loop. An @x:Act command can have zero or more expressions following the tag and optionally a body.

The specific tag determines which expressions are required and optional and how they are to be treated. The specific tag also determines if a body is prohibited, optional, or required. For example, the @x:while command has a test expression following the tag and a body that is executed while the test expression execution returns true.

In some cases multiple commands work together. For example to code an if-then-else, a @x:if command is immediately followed by an @x:else command.

For some @x:Act commands some of the expressions following the tag serve as options. For example, the @x:func command can have visible and lhs options. Option names are not considered to be name uses and are not bound to a corresponding definition.
