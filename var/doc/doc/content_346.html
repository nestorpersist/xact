This iterator takes two other iterators and runs them in parallel.

Function Parallel[ ]

@x:func Parallel[iter1~func,iter2~func](act)
iter1 first iterator function.
iter2 second iterator function.
act the iterator body function. It takes a two parameters, the first for iter1 and the second for iter2.

Semantics

This is an iterator function that iterates over iter1 and iter2 in parallel. It terminates when either of these iterators terminates.
