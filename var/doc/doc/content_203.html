Description

This special function should only be called within Language Definition types. It walks the body of xdom:element tree node. The DocLang example shows a sample use.

Function walk:body

@x:func walk:body(elem~xdom:element,lang)
elem the element whose body is to be walked.
lang The language object to be used to process the body.

Semantics

The pass result. This will be null for bind1 and bind2 and the value produced by evaluating the body in exec.
