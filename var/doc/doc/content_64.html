An @x:Act script can be run from the command line as follows.

cd xact/dev/win

run xxx opt

where xxx specifies the script (and optionally parameters and other information to be passed) and where opt is an optional list of interpreter options.

Note on Windows if any of the command line arguments contain either a & you should enclose that command line argument in double quotes.

On a Macintosh arguments containing & or () must be quoted.
