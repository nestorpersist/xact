Description

This command is used to end execution of the smallest surrounding loop command. Unlike the Java break statement, this command does not immediately exit the loop. Instead, the loop is exited only after completion of the current loop body execution.

Command Syntax

'@x:break'

No attributes, options or body are permitted.

Execution Semantics

The result of executing the @x:break command is null. When the end of execution of the body of the smallest surrounding @x:for or @x:while command is reached, execution of that command is ended with no more body executions.

Examples


   @x:var found := false;
   @x:for i := iter {
      @x:if p(i) {
         "found:"; i;
         found := true;
         @x:break
      } @x:else {
         q(i)
      }
   }
   @x:if ! found {
      "not found"
   }

Note that the First iterator can often be used to avoid the need for break. For example the above example can be rewritten as


   @use library:iterator;
   @x:for i := First[Filter[iter,p]] {
      "found:"; i
   } @x:else {
      "not found"
   }
