Description

This special function should only be called during the exec pass within Language Definition types. It assigns a value to a defined name. The DocLang example shows a sample use.

Function walk:assign

@x:func walk:assign(name~xdom:name,val)~null
name the name that is the target of assignment.
val a value to be assigned.

Semantics

The value of name name is changed to have value val.
