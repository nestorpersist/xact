Description

The Cover function is used to determine which type to use when an infix operation contains operands of different types. For example, for the expression
   2 + 3.7
a call to Cover(int,float) returns float meaning that
   float.sys:Add(2,3.7)
should be called.

Note that a null result from Cover will cause an error when called during processing most infix operators meaning that the particular combination of types is not supported for that operator.

Function Cover

@x:func Cover(a~type,b~type)~boolean
a first parameter
b second parameter

Semantics

If a and b have the same type then the result is
   a
else if HasDot(a,"sys:Cover") and a.sys:Cover(b) returns true, then the result is
   a
else if HasDot(b,"sys:Cover") and b.sys:Cover(a) returns true, then the result is
   b
else the result is
   null
