Description

This command is used for program comments. Comments are not discarded in the @x:Act parser, but instead are kept in the parse tree but have no effect on execution.

Command Syntax

All attributes, options and body are ignored.

Execution Semantics

The result of executing a @x:comment command is null.

Examples


   @x:comment "this is a comment";
   // This is a sugared form of x:comment command
   /* This is another
   sugared form of the x:comment command
   */
   @x:comment "heres how to comment out code" {
      @x:if x == null {
         x := y
      }
   }
