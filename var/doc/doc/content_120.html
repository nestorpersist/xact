Description

This command executes its body only if a test expression is true.

Command Syntax

'@x:if' <exp>

No options are permitted.

Execution Semantics

The attribute expression is evaluated and must yield a boolean result. If the result is true, then the result of executing the @x:if command is the result of the executing its body. If the result is false, then the body is not executed and the result of the @x:if command is null.

Examples


   @x:if i < j {
      "less"
   } @x:orif i > j {
      "greater"
   } @x:else {
      "equal"
   }
