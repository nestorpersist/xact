This iterator is similar to the Nest iterator except the second name paramter is a function that takes the value of the first iterator as a parameter and returns the second iterator.

Function NestF[ ]

@x:func NestF[iter1~func,iter2f~func](act)
iter1 first iterator function.
iter2f a function that takes a single parameter and returns an iterator.
act the iterator body function. It takes a two parameters, the first for iter1 and the second for the result of calling iter2f.

Semantics

This is an iterator function that iterates over iter1 and for each iter1 value iterates over all iter2f values.

Examples


   @x:func rng[high](low) { low..high }
   @x:for [i,j] := NestF[1..3,rng[3]] {
      i; ":"; j; eol
   }
