Description

This function is use to encode information that is to be passed from one script to another via a URL. It is called by the library A and FORM functions.

Function XMLUnparseQuery

@x:func XMLUnparseQuery(tree~xdom:call)~string
tree The information to be passed. The call should have name query and one or more parameters. The first parameter is typically another xdom:call that gives the script to be called and its parameters (both positional and named). Parameters value must be strings. The remaining parameters are trees for expressions of the form name=val where val is a string. The called scripts can access these values via passed.

Semantics

The encoded version of tree.
