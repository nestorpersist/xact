Description

The thread type is used to represent threads and is based on the Java Thread type. Threads are created with the x:thread command.

Type thread

@x:type thread

@x:type @x:func sys:Eq(a~func,b~func)~boolean
@x:type @x:func sys:Ne(a~func,b~func)~boolean
@x:self @x:func toString()~string

Type Function sys:Eq

@x:func sys:Eq(a~func,b~func)~boolean
a first parameter
b second parameter

Semantics

If a and b are the same thread, then the result is true, else the result is false.

Type Function sys:Ne

@x:func sys:Ne(a~func,b~func)~boolean
a first parameter
b second parameter

Semantics

If a and b are the same thread, then the result is false, else the result is true.

Self Function toString

@x:func toString()~string

Semantics

The result is the name of the thread.
