@x:Act has a single grammar that describes both lexical and context-free syntax. The grammar is specified using an extended BNF.

The complete grammar can be found in Combined Syntax. The grammar can also be browsed starting with the <script> root non-terminal.

The grammar is ambiguous for <exp>s with <prefix op>s and <infix op>s. This is resolved with a Precedence table.

Each @x:Act script will be a single <script> contained in a file with extension .xact.
