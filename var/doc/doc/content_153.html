Description

View of xdom:element that allows unnamed attributes to be treated as an array. Array values are stored as follows.

string value stored in a xdom:string node
int value stored in a xdom:int node
float value stored in a xdom:float node
any other value stored in a xdom:value node

View array

@x:view array

@x:view @x:func sys:Is(v~boolean)~boolean
@x:view @x:func sys:Make(%L)~array

View Function sys:Is

@x:func sys:Is(v~boolean)~boolean
v value to test

Semantics

The result is v?xdom:element.

View Function sys:Make

@x:func sys:Make(%L)~array
%L a list of unnamed values or a list or two parameters: the first of form size=n and the second a unnamed value

Semantics

If a list of values is specified, the result is a new array constructed from the specified list of values. For array(size=n,v), the result is a new array of size n with each element initialized to v. The tag of the resulting xdom:element is "array".
