The @x:Act files have the following directory structure.

  • xact/dev. Source file directories.
    • xact/dev/src. Java source files.
    • xact/dev/script. @x:Act scripts.
    • xact/dev/script/example. Examples includes in documentation.
    • xact/dev/script/library. @x:Act libraries.
    • xact/dev/script/xad. These scripts are used to generate the @x:Act documentation. These scripts and their input format are not documented and error checking is limited, so use at your own risk.
    • xact/dev/script/xad/src. The orignal source files from which the @x:Act documentation is generated. Changes to @x:Act documentation are made to these files not to the derived HTML files.
    • xact/dev/win. Windows scripts and commands for building and running @x:Act.
    • xact/dev/mac. Mac OSX scripts and commands for building and running @x:Act.
  • xact/var. Derived file directories. Files in these directories are are generated and should never be directly modified. Section Building describes how they are built.
    • xact/var/doc/doc. @x:Act documentation. Open index.html in a web browser.
    • xact/var/javadoc. Javadoc for the @x:Act Java code. Open index.html in a web browser.
    • xact/var/logs. Log files from running the @x:Act interpreter from the command line.
    • xact/var/weblogs. Log files from running the @x:Act interpreter from a web server.
    • xact/var/object. Class files from compiling the Java source files.
    • xact/var/xact.jar. The Java jar file that contains the @x:Act interpreter.
