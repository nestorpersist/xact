This iterator takes an iterator and combines its sequence of values.

Function Accumulate[ ]

@x:func Accumulate[iter~func,start,join~func](act)
iter iterator function.
start an initial value.
join a function that takes two parameters.
act the iterator body function. It takes a one parameter.

Semantics

If iter produces values 1, 2, 3 then the result will be join(join(join(start,act(1)),act(2)),act(3))
