View that supports enumerated values.

View enum[ ]

@x:view enum[%values]

%values A list of values.

View Function sys:Is

@x:func sys:Is(v~string)~boolean
v value to test

Semantics

The result is true if v is in the list of values.

View Function sys:Eq

@x:func sys:Eq(a~enum[%values],b~enum[%values])~boolean
a first parameter
b second parameter

Semantics

Result is a == b.

View Function sys:Ne

@x:func sys:Ne(a~enum[%values],b~enum[%values])~boolean

b second parameter

Semantics

Result is a != b.

Semantics

View Function sys:LessEq

@x:func sys:LessEq(a~enum[%values],b~enum[%values])~boolean
a first parameter
b second parameter

Semantics

Result is a <#enum[%values] b | a ==#enum[%values] b.

View Function sys:Greater

@x:func sys:Greater(a~enum[%values],b~enum[%values])~boolean
a first parameter
b second parameter

Semantics

Result is true if a follows b in the list of values.

View Function sys:GreaterEq

@x:func sys:GreaterEq(a~enum[%values],b~enum[%values])~boolean
a first parameter
b second parameter

Semantics

Result is a >#enum[%values] b | a ==#enum[%values] b.

Examples


   @x:const color := enum["red","green","blue"];
   @x:var c~color := "red";
   ~c <#color "green";
   ~c > "green"
