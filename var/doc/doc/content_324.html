This function permits an arbitrary string value to be correctly displayed on an XHTML page.

Function quoteXML

@x:func quoteXML(s~string)~string
s a string value.

Semantics

the value of s escaped to display as HTML. This includes escaping special HTML characters <, >, and &. It also changes spaces to nonbreaking spaces and inserts <br/> tags at the end of lines.
