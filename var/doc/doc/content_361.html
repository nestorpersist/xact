This library defines a set of types for common units. It instantiates the general Units type for these specific units. An example of its use can be seen in the utest example. The base units defined are the following.

  • meter. a unit of distance.
  • gram. a unit of mass.
  • second. a unit of time.
  • ampere. a unit of electrical current.
  • kelvin. a unit of temperature.
  • candela. a unit of light intensity.
  • mole. a unit of substance.
  • radian. a unit of angle.
  • steradian. a unit of solid angle.
  • bit. a unit of information.
  • dollar. a unit of currency.

These base units plus other scalings (foot versus meter) and derived units (like watts) are defined by the code shown in the next section.
