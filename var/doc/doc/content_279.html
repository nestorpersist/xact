Cover rules permit mixed type arithmetic. Consider for example a + b. If a and @icode{b}/> have the same type then the sys:Add operation for that type is used. If however a and b have different types, then cover rules determine which of those two types to use.

Consider for example, 2 + 3.5. Since the type float covers int, the float.sys:Add operation is used. Cover rules are given by sys:Cover type operations. For the example here, float.sys:Cover(2) returns true.
