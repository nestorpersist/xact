This iterator takes an iterator, a filtering function, and a filter value and iterates over all values that satisfy the filter.

Function FilterVal[ ]

@x:func FilterVal[iter~func,filter~func,fval](act)
iter iterator function.
filter filter function. Takes two parameters and returns a boolean result.
fval a value.
act the iterator body function. It takes a one parameter.

Semantics

This is an iterator function that iterates over over all values v of iter where filter(v,fval) returns true.
