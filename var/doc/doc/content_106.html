Syntax
<xml> ::= <element> | <escape>
<xml> ::= text - {text {'<' | '&'} text}
<xml> ::= '<!--' {text - {text '-->' text}} '-->'

Description

This is all the forms that are allowed inside an XML <element>.

Parsing

Text here is represented as a xdom:string tree node with type "xml".

An escaped entity is represented as a xdom:name tree node with kind "xml".

A XML comment is represented as a xdom:string tree node with kind "comment".

Execution

Execution of XML text results in a rendering of that text. Three special XML characters are escaped: < becomes &lt;, > becomes &gt;, and & becomes &amp;. Names are replaced with the string value of the name. These name strings are also escaped for the three special XML characters.
