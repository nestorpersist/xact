This function is used to generate an xdom:element node that is an HTML a tag that represents a call to another @x:Act script. This call often appears as attribute of a @x:tag command.

Function A[ ]

@x:func A[path~string,pass=pass1~rec := rec(),inter=inter1~string := options.inter#rec,prefix=prefix1~string := ""](%args)
path the script path of the script to be called.
pass=pass1 (optional) additional values to be passed in the call.
inter=inter1 (optional) the name of the @x:Act interpreter.
prefix=prefix1 (optional) a string to insert at the beginning of the generated URL.
%args a list of string parameters to be passed.

Semantics

Examples

The following command

@x:tag A[ABS.a:b,inter="xx",pass=rec(a="3",b="false")]("a","b","d e+f") { "test ab" }

produces the following output

<a href="xx?a:b(a,b,d+e!(=43)f)&amp;a=3&amp;b=false">test ab</a>
