This func is used to relink a Graphs output by Unparse using the graph=true option. See the graph example to see how this works.

Function graphExpand

@x:func graphExpand(t~xdom)~xdom
t The result of parsing the output of Unparse with graph=true.

Semantics

Result is the linked graph: a modified version of t with graph defs and uses replaced by links. Note the graph is modified in place, not copied.
