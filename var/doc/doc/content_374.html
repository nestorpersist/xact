Function Vals[ ]

@x:func Vals[t~xdom](act)
t an xdom tree.
act the iterator body function. It takes a two parameters, the first of type enum["before","between","after"] and the second of type xdom.

Semantics

This is an iterator function walks the tree in standard order. For every node in the tree the body function act is called twice, once when a node is first visited ("before") and once when the node is about to be left ("after"). It is called an additional time ("between") for xdom:element (after the attributes but before the body) and for xdom:call (after the function but before the parameters).
