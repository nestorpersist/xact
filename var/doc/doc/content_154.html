Description

View of xdom:element that provides access to the list of attributes.

View Function sys:Is

@x:func sys:Is(v~boolean)~boolean
v value to test

Semantics

The result is v?xdom:element.

Self Const size

@x:const size~int

Semantics

The number of attributes in the element.

LHS/RHS Self Function sys:Subscript

@x:func sys:Subscript(index~int)~xdom
index The index of the attribute to be selected. For element e this must be in the range 1..e.size#attr.

Semantics

The indexth attribute.

Self Function insert

@x:func insert(val~xdom,pos~int := self.size#attr)~void
val a value to be inserted
pos (optional) this must be in the range 1..self.size#attr.

Semantics

Inserts the new attribute value at the specified position. If the position is 0, the value is inserted at the beginning of the attribute list. Otherwise, the value is inserted after the pos attribute.

Self Function delete

@x:func delete(pos~int)~void
pos (optional) this must be in the range 1..self.size#attr.

Semantics

The attribute at the specified position is deleted from the attribute list.

Self Const all

@x:const all~func

Semantics

An iterator function that iterates over all the xdom attributes.
