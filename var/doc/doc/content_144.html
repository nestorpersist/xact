Description

The type view is the type of all views. Views are declared using the @x:view tag. Objects of a type are constructed using the constructor form of <qualified> expressions.

Type view

@x:type view

@x:type @x:func sys:Eq(a~view,b~view)~boolean
@x:type @x:func sys:Ne(a~view,b~view)~boolean
@x:self @x:func toString()~string

Type Function sys:Eq

@x:func sys:Eq(a~view,b~view)~boolean
a first parameter
b second parameter

Semantics

If a and b are the same view, then the result is true, else the result is false.

Type Function sys:Ne

@x:func sys:Ne(a~view,b~view)~boolean
a first parameter
b second parameter

Semantics

If a and b are the same view, then the result is false, else the result is true.

Self Function toString

@x:func toString()~string

Semantics

The result is a string that represents the view. It includes the view name. If the view name includes an extension, extension value strings are included.

Examples


   @x:view v;
   ~Type[v] == view;
   ~Type[v()] != v
