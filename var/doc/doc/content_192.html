Description

This function is the build-in @x:Act parser. It will parse @x:Act scripts and the subset of XML supported by @x:Act syntax.

Function Parse

@x:func Parse(text~string,handler=handler1,fileName=fileName1~string,scriptName=scriptName1~string,name=name1~string)~xdom
text input text to be parsed. This parameter is a Stream Sink. When optimization is on, the implementation will in some cases start up a separate thread to evaluate this parameter.
handler=handler1 (optional) parse event handler object.
fileName=fileName1 (optional) if the text parameter is a file, the name of that file. This is used for error reporting. Only one of fileName, scriptName, and name may be present.
scriptName=scriptName1 (optional) if the text parameter is a script file, the script name of that script. This is used for error reporting. Only one of fileName, scriptName, and name may be present.
name=name1 (optional) a short description of the text being parsed. This is used for error reporting. Only one of fileName, scriptName, and name may be present.

Semantics

Returns the resulting parse tree.

If the handler is specified it may have any of the following operations.

Self Function endAttr

@x:func endAttr(attr~xdom:element,attr~xdom)~TypeOr[void,xdom]
attr element that contains the attribute.
attr attribute.

Semantics

Called at the end of an attribute. A result of null discards the attribute. Returning attr keeps the attribute. Returning any other value substitutes that value for the attribute.

Semantics

Called at the end of a child element. A result of null discards the child. Returning attr keeps the child. Returning any other value substitutes that value for the child.
