Every command (<simple cmd> and <compound cmd>) is a scope. Since commands can be nested, scopes can be nested.

Each <script> is also a scope and encloses all other scopes within the script.

There is a single special built-in scope that encloses all script scopes. Build-in names are defined in the built-in scope.
