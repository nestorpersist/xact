This iterator takes two other iterators and runs them in sequential order.

Function Seq[ ]

@x:func Seq[iter1~func,iter2~func](act)
iter1 first iterator function.
iter2 second iterator function.
act the iterator body function. It takes a one parameter.

Semantics

This is an iterator function that iterates over iter1 and then over iter2.
