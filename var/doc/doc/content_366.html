Type Units[ ]

@x:type Units[%dim]

%dim an array of strings, one for each base unit.

@x:type @x:func baseUnit(uname~string)
@x:type @x:type Unit[%DIM]

Type Function baseUnit

@x:func baseUnit(uname~string)
uname the name of one of the base units specified in dim.

Semantics

Within the type Units["a","b","c"] the result of baseUnit("b") will be Unit[a=0,b=1,c=0].

Type Function sys:Add

@x:func sys:Add(a~Unit,b~Unit)~Unit
a first parameter
b second parameter

Semantics

Both a and b must have the same type. This type will be the type of the result. The result is the sum of a and b.

Type Function sys:Sub

@x:func sys:Sub(a~Unit,b~Unit)~Unit
a first parameter
b second parameter

Semantics

Both a and b must have the same type. This type will be the type of the result. The result is the value from subtracting b from a.

Type Function sys:Minus

@x:func sys:Minus(a~Unit)~Unit
a first parameter

Semantics

The result will have the same type as a. The result value will be the negation of a.

Type Function sys:Mult

@x:func sys:Mult(a~Unit,b~Unit)~TypeOr[Unit,float]
a first parameter
b second parameter

Semantics

The result type will be the sum of the exponents of a and b. If all exponents are 0 then the result will have type float. The result is the product of a and b.

Type Function sys:Div

@x:func sys:Div(a~Unit,b~Unit)~TypeOr[Unit,float]
a first parameter
b second parameter

Semantics

The result type will be the result of subtracting b exponents from a exponents. If all exponents are 0 then the result will have type float. The result is the value from dividing a by b.

Type Function sys:Eq

@x:func sys:Eq(a~Unit,b~Unit)~boolean
a first parameter
b second parameter

Semantics

Both a and b must have the same type. If a and b have the same value, then the result is true, else the result is false.

Type Function sys:Ne

Semantics

Self Function sys:Subscript

@x:func sys:Subscript(val~TypeOr[int,float])~Unit

Semantics

Used to specify a value with units. For example, the value 10 feet would be encoded as foot(10) which is equivalent to 10 * foot.

Self Function in[ ]

@x:func in[]~float
t

Semantics

The value of self and t must have the same Unit type. The result of meter(3).in[centi[meter]] is equivalent to meter(3) / centi[meter] whose result value is 300.0.
