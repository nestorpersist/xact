Description

The double star operator by convention is used when unparsing an xdom:value tree nodes. Such nodes are for internal use only and when unparsed, the double star operator serves as a place holder.

Semantics

The form
   **a
is equivalent to
   XDOMValue(a)

The form
   **#ba
is equivalent to
   b.sys:XDOMValue(a)
