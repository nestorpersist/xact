The @x:Act interpreter contains a early version of an @x:Act debugger. The debugger is incomplete and has many rough edges and bugs.

The debugger is entered when events specified by interpreter options occur.

  • debug. Debug when the invoked script or function is about to be executed.
  • debugAll. Debug when any invoked or used script is about to be executed.
  • debugError. Debug when any error occurs.
  • debugWarn. Debug when any warning or error occurs.

There are two single-step control panels: 3x3 and 2x2. The 3x3 columns are break before, break before or after, break after. The 3x3 rows are break at higher level, break at current level, and break at lower level. The 2x2 panel includes S (restart), X (break when running), => (continue), <= (rerun expression), Q (quit).
