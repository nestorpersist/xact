This iterator takes an iterator and a filtering function and iterates over all values that satisfy the filter.

Function Filter[ ]

@x:func Filter[iter~func,filter~func](act)
iter iterator function.
filter filter function. Takes a single parameter and returns a boolean result.
act the iterator body function. It takes a one parameter.

Semantics

This is an iterator function that iterates over over all values v of iter where filter(v) returns true.
