This iterator function breaks a string into parts based on a separator string.

For example the string "a,b,c" is separated by "," into "a", "b", and "c".

Function Separate[ ]

@x:func Separate[s~string,sep~string](act)
s a string
sep a separator.
act the iterator body function. It takes a single parameter of type string.

Semantics

This is an iterator function that iterates over all the parts of s as separated by sep.
