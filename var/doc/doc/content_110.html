Description

This command is used to introduce a new nested scope or to group commands.

Command Syntax

'@x:block'

No attributes or options are permitted.

Execution Semantics

The result of executing a @x:block command is the result of executing its body.

Examples


   @x:begin {
      @x:const a := 3;
      a + a
   }
