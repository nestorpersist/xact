Syntax
<compound cmd> ::= <long cmd> | <element>
<compound cmd> ::= {<id> ':'}? '/*' {text - {text '*/' text}} '*/'
<compound cmd> ::= {<id> ':'}? '//' {text - {text eol text}} eol

Description

A compound command can be a tagged log command, and XML element, or a comment.

Parsing Long Comments

Comments are a form of syntactic sugar. The following form

/*...*/

is equivalent to

Comment("...")

and will produce an xdom:call tree node with kind set to "long".

The form

z:/*...

is equivalent to

@z:comment "..."

and will produce an xdom:element tree node with kind set to "long".

Parsing Short Comments

Comments are a form of syntactic sugar. The following form

//...

is equivalent to

Comment("...")

and will produce an xdom:call tree node with kind set to "short".

The form

z://...

is equivalent to

@x:comment "..."

and will produce an xdom:element tree node with kind set to "long".
