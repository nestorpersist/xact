Description

The Chars iterator iterates over each character of a string.

Function Chars[ ]

@x:func Chars[str~string](act)
str a string to be iterated over. This parameter is a Stream Sink. When optimization is on, the implementation will in some cases start up a separate thread to evaluate this parameter.
act the iterator body function. It takes a single parameter of type int which is a character code.

Semantics

This is an iterator function that iterates over all character of a string.

Examples


   @x:for ch := Chars["abc"] {
      f(ch)
   }
