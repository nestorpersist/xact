The following are all stream sources.

  • The result of the file read function.
  • The result of stream receive function.
  • The result of the ++ infix operation.
  • The result of joins (see Join Rules). For example the result of
       @x:block {e1;e2}
    will be a source that first does e1 then does e2. If e1 is a source then it is also recursively streamed. Same of e2. Joins from commands such as @x:if and @x:while are also streamed.
