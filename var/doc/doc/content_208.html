Description

The expression
   **a
is desugared to
   XDOMValue(a)

The Unparse function uses the ** prefix operator to represent xdom:element node value that are for internal use only. This output can be useful in debugging.

Function XDOMValue

@x:func XDOMValue(a)
a first parameter

Semantics

It is an error to execute a call to XDOMValue.
